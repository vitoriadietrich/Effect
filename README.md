# Animated Navigation Menu Bar Hover Effect in HTML and CSS

Um projeto simples de **menu de navegação com efeito hover animado** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **menu de navegação moderno com animações no hover**, deixando a interface mais interativa.

## 🛠️ Tecnologias

- HTML
- CSS (transition, hover effects)

## 🎯 Funcionalidades

- Animação ao passar o mouse
- Menu de navegação estilizado
- Design simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
